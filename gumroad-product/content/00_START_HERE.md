## Your Quick-Start (Do This First)

### Step 1: Get a Notion Account (Free)
If you don't have one, go to notion.so and sign up. It's free.

### Step 2: Import the Workspace Template
1. Open the `notion-template/` folder in this download
2. Click the template link inside
3. Click "Duplicate" in Notion
4. Rename it to "[Your Business] Automation Hub"

### Step 3: Pick ONE Automation to Start
Don't try to automate everything. Pick the ONE thing that wastes the most time.

Common starting points:
- **Lead follow-up** (if you have incoming leads)
- **Email triage** (if your inbox is overwhelming)
- **Social posts** (if you struggle to post consistently)

### Step 4: Follow a Recipe
Find your automation type in the `recipes/` folder. Follow the steps exactly for your first week.

## How the Recipes Work

Each recipe has 4 parts:
1. **What it does** — plain English description
2. **Why it works** — the business logic
3. **The exact prompt** — copy this into ChatGPT/Claude
4. **Setup steps** — numbered actions to get it running

Don't edit the prompts until you've run the automation at least 3 times. The prompts work — trust the recipe first.

## How the Prompts Work

The prompts in `prompts/` are organized by use case. Each prompt:
- Is written for ChatGPT or Claude (works with both)
- Has a specific format to get consistent output
- Includes an example so you can see what good looks like

To use: copy the prompt, fill in [your specifics], paste into AI, done.
