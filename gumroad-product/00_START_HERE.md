## Your 3-Day Quick-Start

### Day 1: Set Up (30 minutes)
1. Create a free Notion account at notion.so
2. Open the Notion template link (in the NOTION-TEMPLATE.md file)
3. Click **Duplicate** to add it to your workspace
4. Rename it to "[Your Business] Automation Hub"
5. Set up ChatGPT (free account at chat.openai.com)

### Day 2: Run Your First Automation (20 minutes)
1. Pick ONE pain point from your business
2. Find the matching recipe (50 AUTOMATION RECIPES.md)
3. Copy the prompt exactly
4. Paste it into ChatGPT
5. Use the output — that's it

### Day 3: Build the Habit (15 minutes)
1. Run Morning Inbox Triage for the first time
2. Generate your first week's social content
3. Review what worked
4. Set a recurring time to run your automation
